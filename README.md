# 📚 Library Management System

A simple and interactive **Library Management System** built using **Python** and **Streamlit**.
This project allows users to manage books in a library with features like adding books, issuing books, returning books, and viewing available or issued books through a clean web interface.

---

## 🚀 Features

* ➕ Add new books with quantity
* 📖 Issue books to students
* 🔄 Return issued books
* 📚 View available books
* 👨‍🎓 View issued books records
* 🎨 Interactive Streamlit UI
* 💾 Session-based data storage using `st.session_state`

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/library-management-system.git
```

### 2️⃣ Move into Project Folder

```bash
cd library-management-system
```

### 3️⃣ Install Streamlit

```bash
pip install streamlit
```

---

## ▶️ Run the Project

```bash
streamlit run streamlit_library_app.py
```

After running the command, Streamlit will open the project in your browser automatically 🚀

## 📸 Project Functions

### 📚 Add Book

Add books along with quantity to the library.

### 📖 Issue Book

Issue available books to students.

### 🔄 Return Book

Return issued books back to the library.

### 📋 Show Books

Display all available books and quantities.

### 👨‍🎓 Issued Books

View records of issued books.
